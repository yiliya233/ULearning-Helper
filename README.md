# ULearning Helper

Edge/Chrome 浏览器扩展，针对优学院**知识图谱**模块，自动识别考试/作业题目，调用 AI 接口获取答案并自动填写提交。

## 功能特性

- 自动识别题目类型（单选、多选、判断、填空、简答）
- 调用 OpenAI 兼容 API 获取答案
- 自动填写并提交
- 答案记忆：提交后记录正确答案，重考时直接使用
- 判断题智能翻转：答错后自动翻转答案
- 章节自动导航：答完一章自动进入下一章
- 悬浮窗控制面板，支持拖拽和折叠
- 可中断：随时点击停止按钮

## 安装步骤

1. 下载或克隆本仓库
2. 打开 Edge 浏览器，地址栏输入 `edge://extensions/`
3. 打开右上角「开发者模式」
4. 点击「加载解压缩的扩展」
5. 选择本项目文件夹
6. 插件安装完成，点击工具栏上的插件图标

## 配置

点击插件图标后，在弹出窗口中填写：

| 字段 | 说明 | 示例 |
|------|------|------|
| API地址 | OpenAI 兼容接口地址 | `https://api.openai.com/v1/chat/completions` |
| API Key | 你的 API 密钥 | `sk-xxxx...` |
| 模型名称 | 使用的模型 | `gpt-4o-mini` |

填写完成后点击「保存配置」，建议先点击「测试API连接」确认正常。

## 使用方法

1. 进入优学院知识图谱的考试或作业页面
2. 点击插件图标，打开悬浮窗控制面板
3. 点击「开始」按钮
4. 插件会自动：
   - 点击「去测验」进入答题
   - 逐题识别、调用 AI、填写答案、提交
   - 记录正确答案用于重考
   - 答完一章自动进入下一章

## 支持的题型

- 单选题
- 多选题
- 判断题（对/错）

## 注意事项

- 需要自行准备 OpenAI 兼容的 API Key
- AI 回答可能存在错误，插件会在答错时自动重考
- 首次使用建议先用少量题目测试
- 插件依赖页面 DOM 结构，优学院更新后可能需要调整选择器

## 自定义适配

如果优学院页面结构与默认不同，可以在设置页面（点击悬浮窗齿轮图标）中自定义选择器：

- 题目容器选择器
- 选项选择器
- 提交按钮选择器等

## 技术栈

- Manifest V3 (Chrome/Edge 扩展)
- 原生 JavaScript（无框架依赖）
- OpenAI 兼容 API
